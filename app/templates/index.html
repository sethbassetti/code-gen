Model Info

This project was built using GPT-2, a transformer built by Open-AI. Specifically, a transformer is a sequence-sequence large-scale language model using self-attention layers to learn relationships between various parts of text.

Training Data

An example of where you can put an image of a project, or anything else, along with a description.

Evaluation

The output of this model was intrinsically qualitatively judged based on the coherence of the output text. Although further quantitative analysis is possible through measures such as n-gram or BLEU metrics, manually judging the quality of text suited the scope of this project.
